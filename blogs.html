Short Dissertation on the Quirks of Pre-Calc

"Math is confusing!" I hear it all the time from multiple people. I finally see where they are coming from.

Last night while doing pre-calc review problems for my calculus class, I stumbled upon this equation: f(t) = (4 - t^2)/(2 - t). A rational function but despite its ungainly appearance, it's not really too hard. Right? Well, yes. Sometimes.

I believe everyone would agree with me that this problem can be simplified and that this is done in the following manner:

f(t) = (4 - t^2)/(2 - t)

f(t) = ((2 - t)(2 + t))/(2 - t)

f(t) = (2 + t)

Makes sense right? (4 - t^2)/(2 - t) = (2 + t). Looks good, but is it really? Let's do some domain and range to find out.

Let's consider f(t) = (2 + t):

Linear function. No specified domain limits, thus it has none. Domain is: (-∞, ∞).

That was simple. So if (4 - t^2)/(2 - t) = (2 + t) then shouldn't the domain of f(t) = (4 - t^2)/(2 - t) also be (-∞, ∞)? One would think so. However, a closer examination proves otherwise:

Rational function. No specified domain limit, however because of the rational nature, there must be. Calculated:

(2 - t) > 0 or (2 - t) < 0 so that denominator is != 0.

2 - t > 0 or 2 - t < 0

2 > t or 2 < t

t < 2 or t > 2

Domain is then: (-∞, 2) U (2, ∞)

If you were to graph both of these mathematically equal equations, you would get the same line albeit the second equation we found the domain for would have a hole at 2. This is NOT the same as a line that continues on forever. The equations are the same yet not the same. Strange but true.
